## 如何最佳的让两个整数交换数值

```js
// 常规的办法：
let a = 1,
  b = 2;
a += b;
b = a - b;
a -= b;
// 缺点也很明显，整型数据溢出，对于32位字符最大表示数字是2147483647，如果是2147483645和2147483646交换就失败了。

// 黑科技办法：
a ^= b;
b ^= a;
a ^= b;

// ES6 写法
[a, b] = [1, 2];
[b, a] = [a, b];
```
